# Basic AI

This folder contains an introductory overview of basic AI concepts and a small in-browser demo you can open in your browser.

Files
- `index.html` — short guide + an interactive demo using TensorFlow.js (runs in the browser).

How to view
- Open `basic-ai/index.html` in a web browser (or host with GitHub Pages) to see the demo.
